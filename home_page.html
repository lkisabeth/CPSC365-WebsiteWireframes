Notes:

1) Any pending friend requests for the currently logged in user will be displayed in this section (Left side of screen).

1a) The requester's username will be retrieved from the database and displayed here.

1b) These buttons are used to accept or reject the pending friend request. The request card will be removed from the view upon completion.

2) The 10 most recent comments from the user's friends will be displayed here (left side of screen, below Friend Requests)

2a) Information about the commenter will be retrieved from the database and displayed here. Also, the add friend button allows the currently logged in user to add the commenter as a friend.

2b) The total number of likes for each comment will be displayed here and the user will be able to click the "Like" button to add or remove a like.

3) The 10 most recently add movies will be displayed here (right side of screen)

3a) The movie's poster will be displayed here. Users can click the poster to be taken to the movie's detail page.

3b) The movie title will also be clickable and take users to the movie's detail page.

3c) All movie attributes will be displayed in this section.
